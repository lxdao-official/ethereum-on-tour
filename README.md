# 🚀 Ethereum On Tour - 以太坊高校行

## 📚 仓库介绍

本仓库用于收集和整理以太坊高校行活动中，各高校学生提交的 Workshop 学习笔记、实践代码和项目成果。

## 📝 参与流程

### Step 01：Fork 本仓库 🍴

点击仓库页面右上角的 **Fork** 按钮，将仓库 Fork 到你自己的 GitHub 账户下。

### Step 02：创建个人 Workshop 文件夹 📁

每个学生需要在仓库根目录下创建一个独立的文件夹，文件夹命名规则为：**学校名称+GitHub昵称**

例如：
- 北京大学+alice
- 清华大学+bob

在你的个人文件夹中，可以按照以下结构组织内容：

```
学校名称+GitHub昵称/
├── README.md          # 个人简介和学习总结
├── workshop1/         # 第一个 Workshop 的笔记或者代码
│   ├── notes.md      # 详细笔记
│   ├── code/         # 实践代码
│   └── assets/       # 图片等资源
└── workshop2/         # 第二个 Workshop 的笔记或者代码 
    └── ...
```

### Step 03：创建 Pull Request (PR) 📤

1. 访问你 Fork 后的仓库页面
2. 点击 **Compare & pull request** 按钮
3. 填写 PR 标题和描述，说明你提交的内容
4. 点击 **Create pull request** 按钮提交

提交并且在 review 通过后，你的 PR 会被 merge 到仓库 ✅。

## 💡 注意事项
- ✅ 请确保你的笔记内容真实、准确，不包含抄袭内容
- 📝 提交 PR 时，请使用清晰的标题和描述
- ❓ 如有问题，请在 Issues 中或者活动群中提出
